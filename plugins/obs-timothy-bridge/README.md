# obs-Timothy Bridge

Obsidian 插件，用于在 Obsidian 里用傻瓜式入口创建 Codex 任务。

## 常用命令

- `Codex: 问一下`
- `Codex: 优化当前笔记`
- `Codex: 整理当前笔记结构`
- `Codex: 补充当前笔记关联`
- `Codex: 查看最新结果`

## 使用方式

1. 在 Obsidian 中打开命令面板
2. 搜索 `Codex`
3. 选择一个命令
4. 插件自动在 `_Codex/Inbox` 生成任务卡
5. 回到 Codex，说：`处理 Obsidian 任务`

## 本机命令

插件支持执行本机命令，但默认关闭。

只有在插件设置中开启 `允许执行本机命令` 并配置命令后，才会执行。
