# obs-Timothy Bridge

Obsidian plugin for creating `_Codex` bridge tasks and optionally running a local Codex command.

## Features

- Adds a ribbon entry for the Codex command note
- Adds commands for creating bridge tasks
- Creates `_Codex/Inbox`, `_Codex/Results`, `_Codex/Archive`, and `_Codex/Logs`
- Can optionally execute a configured local command

## Safety

Shell execution is disabled by default. Enable it only when you trust the command.

The plugin does not permanently delete notes by itself. Deletion policy remains governed by the bridge protocol.
