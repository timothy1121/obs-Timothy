# Note Types and Structures

## 正式规则

### 1. 对象类

适用：概念、术语、人物、事件

- 概念解释
- 关键特征
- 背景 / 原理
- 关联知识

### 2. 机制类

适用：原理、机制、关系、变量

- 定义
- 作用机制
- 关键变量 / 关系
- 逻辑
- 关联知识

### 3. 结构类

适用：模型、模式、框架、时间线

标准结构：

- 概念解释
- 核心结构
- 运行逻辑
- 适用场景
- 关联知识

时间线变体：

- 概念解释
- 时间节点
- 关键变化
- 阶段逻辑
- 关联知识

### 4. 执行类

适用：方法、步骤、规则、流程

- 概念解释
- 适用条件
- 核心步骤 / 核心流程
- 关键规则 / 注意点
- 关联知识

### 5. 证据类

适用：实验、案例、数据、指标

实验：

- 实验过程
- 实验结果
- 结论
- 关联知识

案例：

- 背景
- 关键过程
- 结果
- 启示
- 关联知识

数据 / 指标：

- 定义
- 计算口径 / 观察方式
- 用途
- 局限
- 关联知识

### 6. 表达类

适用：书摘、摘录、命题、结论、假设、书籍

书籍：

- 概念解释
- 核心观点
- 原理
- 逻辑
- 关联知识

摘录：

- 原文
- 核心意思
- 我的理解 / 用途
- 关联知识

命题：

- 命题内容
- 成立条件
- 推导逻辑
- 关联知识

结论：

- 结论
- 支撑依据
- 适用边界
- 关联知识

假设：

- 假设内容
- 前提条件
- 待验证点
- 关联知识

## 非正式层

### 7. 素材

用于摘录、提纲、片段、灵感、未完成内容。  
不升格为正式知识卡，也不参加模板考核。
