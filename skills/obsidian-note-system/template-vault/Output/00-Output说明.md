# Output

## 定义

Output——输出：不是简单表达，而是产出有价值的、可以重复使用的产品。

## 这里放什么

- 成型成果
- 可复用方案
- 工具化结果
- 产品化内容
- 可交付文档
- 可复用工作流
- 能直接再次调用的输出物

## 这里不放什么

- 还只是原始素材的记录
- 仍在中间组织阶段的过程稿

## 与另外两层的关系

- `Output` 是系统的结果层
- 它承接 `Internalization` 的整理结果
- 输出后的成果仍然可以反向进入 `Input` 或 `Internalization`，继续迭代

## 当前原则

- 输出要有实际价值
- 输出要可以重复使用
- 输出不等于表达欲
- 输出应尽量形成成果、产品、工具或标准件
- 输出尽量能被再次调用，而不是一次性文本
