# 证据类模板

## 适用

实验、案例、数据、指标。

### 实验

**实验过程**  
说明实验如何设计、如何进行。

**实验结果**  
说明观察到了什么结果。

**结论**  
说明实验结果支持什么判断。

说明案例发生的情境和问题。

**关键过程**  
说明采取了什么做法。

**结果**  
说明最后出现了什么结果。

**启示**  
说明这个案例能说明什么。

### 数据 / 指标

**定义**  
说明这个数据或指标是什么。

**计算口径 / 观察方式**  
说明怎么统计、怎么判断。

**用途**  
说明它用来衡量什么。

**局限**  
说明它不能说明什么，或容易产生什么误解。
