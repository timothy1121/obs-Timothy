# 笔记分类与模板索引

这个文件夹用于放最小知识单元、素材和正式原子卡。

## 使用顺序

1. 先判断这条笔记属于哪一类
2. 再打开对应模板
3. 如果内容还不稳定，先保留为 `素材`

## 正式层：6 类母模板

1. `对象类`
2. `机制类`
3. `结构类`
4. `执行类`
5. `证据类`
6. `表达类`

## 非正式层

7. `素材`

## 判断规则

- “它是什么” -> `对象类`
- “它怎么运作” -> `机制类`
- “它由什么组成” -> `结构类`
- “它怎么做” -> `执行类`
- “它证明了什么” -> `证据类`
- “它表达了什么文本或观点” -> `表达类`
