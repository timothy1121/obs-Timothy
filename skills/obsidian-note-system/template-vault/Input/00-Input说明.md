# Input

## 定义

Input——输入：不是简单收集，而是把学习到的知识化为最小知识单元，作为知识库基础，后可以重复使用。

## 这里放什么

- 素材
- 摘录
- 提纲
- 片段
- 原始记录
- 待提炼知识
- 尚未稳定成型的最小知识单元

## 这里不放什么

- 大段总结后的成品内容
- 任务导向的组合稿
- 已经产品化、工具化、成果化的输出

## 与另外两层的关系

- `Input` 是知识进入系统的入口
- 从这里提炼出的稳定单元，可以进入 `Internalization`
- 还不成熟的内容，继续保留在 `Input`，不强行升格

## 当前原则

- 先保留原意
- 先保留精度
- 不急着总结
- 不急着归纳成统一模板
- 素材优先保留为素材
