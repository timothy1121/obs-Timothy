# 快速开始

这个库是原子笔记系统的母版库复制结果。第一次使用，按下面顺序进入。

## 1. 先看总纲

- [[00-原子笔记总纲]]

明确这套系统的三层结构：

- `Input`
- `Internalization`
- `Output`

## 2. 再看母版说明

- [[01-母版使用说明]]

明确这个库里哪些是模板，哪些不是模板。

## 3. 从 Input 开始

- [[Input/00-Input说明]]
- [[Input/01-笔记分类与模板索引]]

先判断内容属于哪一类：

- 对象类
- 机制类
- 结构类
- 执行类
- 证据类
- 表达类
- 素材

## 4. 选模板落笔

- [[Input/10-对象类模板]]
- [[Input/11-机制类模板]]
- [[Input/12-结构类模板]]
- [[Input/13-执行类模板]]
- [[Input/14-证据类模板]]
- [[Input/15-表达类模板]]
- [[Input/16-素材模板]]

## 5. 做内化

- [[Internalization/00-Internalization说明]]
- [[Internalization/01-内化工作板]]

当知识单元已经够稳定，就围绕目标做组合、连接、重组。

## 6. 做输出

- [[Output/00-Output说明]]
- [[Output/01-输出成果模板]]

当内容已经能形成方案、工具、文档或成果，再进入输出层。

## 使用原则

- 先分类，再写结构
- 一张卡只解决一个核心对象
- 素材不强行升格
- 模板库不承载具体知识内容
